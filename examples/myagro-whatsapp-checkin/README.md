# myagro-whatsapp-checkin

A product research project created with **product-kit**.

## Project Structure

```
.product/           # Product research files
├── users.md         # User types and personas
└── problem.md       # Problem statement
```

## Workflow

1. Define your users: `product-kit users`
2. Frame the problem: `product-kit problem`

## Getting Started

```bash
product-kit users
```
